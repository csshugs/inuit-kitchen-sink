The tabs object is very similar to the pack object.

Markup

<ul class="tabs">
    <li class="tabs__item">
        <a href="#" class="tabs__link">Foo</a>
    </li>
    <li class="tabs__item">
        <a href="#" class="tabs__link">Bar</a>
    </li>
    <li class="tabs__item">
        <a href="#" class="tabs__link">Baz</a>
    </li>
</ul>

.tabs__link can be omitted. It just sets the link to display: block.
